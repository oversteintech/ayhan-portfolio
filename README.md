# Ayhan Uzundal Portfolio Site

Static personal portfolio website.

## Deploy with Vercel
1. Create a GitHub repository named `ayhan-portfolio`.
2. Upload all files in this folder.
3. Go to Vercel and import the GitHub repository.
4. Keep default settings and deploy.
5. Optional: connect `ayhanuzundal.com` as a custom domain.

## Files
- `index.html` - main website
- `favicon.svg` - browser icon
- `robots.txt` - SEO crawler instructions
- `sitemap.xml` - SEO sitemap

Before publishing, update `sitemap.xml`, `robots.txt`, and the Open Graph URL in `index.html` if you use a domain different from `ayhanuzundal.com`.
